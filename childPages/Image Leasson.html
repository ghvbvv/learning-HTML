Image Lessons

Steps in building an Image on a webpage

  • Pick an Image (careful on the file extension in the URL)
  • Record URL, cite it, comment for date access, and record the original dimensions
  • Use blockquote and comment as appropiate
  • Use img-tag with attributes src, alt, title, and style with width-height
